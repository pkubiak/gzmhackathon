    box-shadow

    @mixin box-shadow($x, $y, $blur, $spread, $color) { ... }

    Description

    Mixin creating box-shadow css style with support of all browsers

    Parameters

    parameter Nameparameter Descriptionparameter Typeparameter Default value
    $x

    Horizontal offset

    		Number— none
    $y

    Vertical offset

    		Number— none
    $blur

    The blur radius. The higher the number, the more blurred the shadow will be

    		Number— none
    $spread

    The spread radius. A positive value increases the size of the shadow, a negative value decreases the size of the shadow

    		Number— none
    $color

    The color of the shadow

    		Color— none

    Example

    giving shadow to .foo

    .foo {
  @include box-shadow(5px, 5px, 10px, 10px, #ff0000);
}
